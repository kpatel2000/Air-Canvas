# Air Canvas

You can draw your imaginations just by waving your hand in air.

## Requirements

- Python 3.7.9
- OpenCV
- Mediapipe

## Installation

```bash
pip install mediapipe
```

```bash
pip install opencv-python
```


## How to Run
```bash
python VirtualPaint.py
```
